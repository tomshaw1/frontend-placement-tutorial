Using Git

Git is a tool to track changes in files and allow collaboration between developers. This is done by organising code into repositories or repos, which contain all the code for different services across HMRC.

The primary repos we use as frontend developers are HMRC Design System and HMRC Frontend. It may be good to make note of the larger GOVUK repos, GOVUK Design System and GOVUK Frontend.

Most commands you’ll need to work for HMRC can be done through GitHub’s Desktop app. However, best practice is to learn to use git at the command line, for more control. Frame this cheatsheet and hang it above your desk!
