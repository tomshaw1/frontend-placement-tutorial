Installing Visual Studio Code

To write code, we need an IDE, or code editor. The most popular choice and commonly used in HMRC is Visual Studio Code or VSCode. You can install a different code editor, but this page will get you set up with VSCode and some other helpful extensions.

Get started

Install VSCode

VSCode allows us to install extensions which can help improve workflow. Here are a few recommendations and how to install them:

  1. Open VSCode. On the sidebar, there should be an icon that looks like this:

  2. Open this section, and you will now see a search bar and a list of recommended extensions. You can install any you think will be helpful, but here are a few:

    • ESLint

    • GitLens

    • Nunjucks

    • Docker

Don't worry about understanding how VSCode or the extensions work yet; we'll get to that. For now, move on to looking into Git.
