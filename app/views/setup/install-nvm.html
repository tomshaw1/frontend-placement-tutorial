Using Node and NPM

We use Node for a couple of reasons - it allows us to write Javascript code for the server, which means we can make more persistent additions to websites by saving data to the server.

However, the more commonly interacted with feature of Node is npm or Node Package Manager, which is a way to share and incorporate other peopleâ€™s code into our projects, bundled as packages.

This is a really powerful tool, as we can rely on packages to handle components of a website that may be more challenging, or time consuming to add ourselves. Things like build processes, linting and style helpers can be included to improve workflow.

We also publish our own npm package. To add hmrc-frontend to any project, simply use the following code:

{{ codeSnippet({ code: 'npm install hmrc-frontend' }) }}

How to install Node

To use this functionality, we'll need to install Node. You can just do this by going to the Node website, however it is better to install through NVM.

NVM is an open source tool that can be used to install Node, which allows you to change between Node versions with a simple terminal command rather than installing and reinstalling other versions. To do this, follow the instructions on the NVM GitHub page to get up and running.
