Accessibility

aXe

aXe is an accessibility testing tool, which HMRC includes in their code as a node.js package. We can use it in our test file for components to check that there are no accessibility issues with the pattern we create, which we will do later.

aXe uses WCAG, so we can automatically discover some accessibility issues and keep in line with our AA requirement.
