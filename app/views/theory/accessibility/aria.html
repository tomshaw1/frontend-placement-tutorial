ARIA

Accessible Rich Internet Applications (ARIA) is a set of attributes we can use on the web to help define web content to those with accessibility needs.

In some of HMRCâ€™s patterns, we use ARIA to define a multitude of things to users, such as:

  • The role of different elements
  • Whether a button or link is the currently selected one in a list.
  • Have an element be hidden on the page, but still be read to a screen reader.

We will use some aria attributes when creating our component, so more on this later. For more information on ARIA, use the MDN documentation.
