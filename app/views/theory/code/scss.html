SCSS

What is SCSS?

Sass, or SCSS, is a preprocessor scripting language in place of regular CSS. This is used in HMRC for the features that it has over CSS, such as importing custom CSS files (commonly used to bring in GOVUK styling) and nested syntax.

For the most part, styling a component will feel very similar to using CSS, however, there are some quality of life additions to make your code more readable and efficient.
