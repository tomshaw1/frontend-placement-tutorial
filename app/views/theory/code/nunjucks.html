Nunjucks

What is Nunjucks?

Nunjucks is a templating language that allows us to dynamically create HTML on the web. This is how to display the content on the page in a way that can be configured for use across HMRC in different ways.

For example, we can use a for loop to add rows to a table based on the amount of content we have to add, rather than creating a table statically that has empty cells or holds old information.
